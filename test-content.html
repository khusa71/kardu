JavaScript Programming Guide

Chapter 1: Variables and Data Types

JavaScript variables store data values:

  • let - creates block-scoped variables
  • const - creates constants that cannot be reassigned
  • var - creates function-scoped variables (legacy)
let name = "John";
const age = 25;
var city = "New York";

Chapter 2: Functions

Functions are reusable blocks of code:

function greet(name) {
    return "Hello, " + name + "!";
}

const multiply = (a, b) => a * b;

Chapter 3: Arrays and Objects

Arrays store ordered collections:

let fruits = ["apple", "banana", "orange"];

Objects store key-value pairs:

let person = {
    name: "Alice",
    age: 30,
    city: "Boston"
};
